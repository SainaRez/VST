Envelope

How to use:

In a frame that you want to be an envelope (a container for other frames that can be opened and closed):

  1. instantiate envelope = new Envelope(...) object with a reference to a RealityInterface and other parameters
  2. Use envelope APIs like envelope.open(), envelope.close(), envelope.onFrameAdded(...), ...
  3. To send a message to frames dropped into this envelope, use: envelope.sendMessageToAllContainedFrames({ exampleMessageName: messageData }); (You can also use sendMessageToFrameWithId or sendMessageToFrameAtIndex to send to a specific one)
  4. To listen for messages from contained frames, use: envelope.onMessageFromContainedFrame(function(message) { if (typeof message.exampleMessageName !== 'undefined') { // respond to message.exampleMessageName } });
  5. Ensure that all frames you want to be compatible with this envelope follow the instructions in envelopeContents.js

Envelope

Defines an interface that declares this frame to be an "envelope" frame that can contain other frames to form some form of a relationship between them all. Envelopes have an "open" state, where they take up the fullscreen 2D UI, and a "closed" state, where they are minimized into a small 3D icon in space. Their contained frames are only visible when the envelope is open.

new Envelope(realityInterface: RealityInterface, compatibleFrameTypes: Array<string>, rootElementWhenOpen: HTMLElement, rootElementWhenClosed: HTMLElement, isStackable: (boolean | undefined), areFramesOrdered: (boolean | undefined))
Parameters
realityInterface (RealityInterface) reference to the RealityInterface API object
compatibleFrameTypes (Array<string>) array of types of frames that can be added to this envelope
rootElementWhenOpen (HTMLElement) a containing div that will be rendered when open (fullscreen 2D)
rootElementWhenClosed (HTMLElement) a containing div that will be rendered when closed (small 3D icon)
isStackable ((boolean | undefined)) defaults to false
areFramesOrdered ((boolean | undefined)) defaults to false

FrameData

This contains all the necessary information to keep track of a frame that the envelope contains. More params can be added as necessary for more features.

new FrameData(id: string, type: string)
Parameters
id (string) the frame uuid, used as an address to send messages to it
type (string) the frame type, used to ensure it is a compatible with this envelope or to distinguish between different contained frames' capabilities
